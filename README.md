# 🎬 Lista de Filmes App

Aplicativo desenvolvido em React Native com Expo para demonstrar:

- React Navigation
- Consumo de API REST
- Componentes com React Native Paper
- Navegação entre telas

---

# 📱 Tecnologias utilizadas

- React Native
- Expo
- React Navigation
- React Native Paper
- API REST

---

# 🚀 Funcionalidades

✅ Listagem de filmes  
✅ Tela de detalhes  
✅ Navegação entre páginas  
✅ Tema dark preto e roxo  
✅ Consumo de API REST
